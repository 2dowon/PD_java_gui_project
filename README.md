# JAVA GUI PROJECT - PLAYBOX

## Introduction

PLAYBOX는 [Playdata](https://playdata.io/)에서 진행한 미니 팀프로젝트입니다. Java로 GUI를 구현하는 것이 이번 프로젝트의 조건으로 저희 팀은 Swing을 이용해 **영화 예매 프로그램** 을 개발했습니다.
